Im Hersteller-Werkstattsystem, Code 97DR ; Ein Ausfall des Kondensators kann zu einer Sensorsignalveränderung führen. In den meisten Fällen wird die Sensorsignalveränderung vom Steuergerät erkannt und das Airbagsystem abgeschaltet (Airbagkontrolle leuchtet). Es kann jedoch nicht völlig ausgeschlossen werden, dass das Steuergerät die Sensorsignalveränderung als crashähnliches Signal (fälschliche Frontcrasherkennung) interpretiert, wodurch eine Fehlauslösung des/der Airbags und/oder Gurtstraffer möglich ist. Abhilfe: An den betroffenen Fahrzeugen wird die Software des Airbag Steuergeräts aktualisiert ; Im Hersteller-Werkstattsystem, Code 69Z9 ; Eine fehlerhafte Konfiguration des Body-Control-Modul (BCM) kann dazu führen, dass ein Ausfall der vorderen LED-Fahrtrichtungsanzeiger nicht erkannt und nicht angezeigt wird. Abhilfe: Ein Softwareupdate wird durchgeführt.
